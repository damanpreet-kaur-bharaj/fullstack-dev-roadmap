# Module 01 — HTML + CSS

## Key Concepts Covered

### HTML
- Semantic elements — header, nav, main, section, article, footer
- Forms — input types, labels, validation attributes
- Accessibility — alt text, aria labels, semantic structure

### CSS
- Box model — margin, padding, border, content
- Flexbox — display flex, justify-content, align-items, gap
- CSS Grid — grid-template-columns, grid-template-rows
- CSS Variables — --primary-color, --font-size-base
- Responsive design — media queries, mobile first approach
- BEM naming — .block__element--modifier
- Transitions and animations

## Key Rules I Learned
- Always use semantic HTML — not just divs
- Mobile first — write CSS for small screens first
- Use CSS variables for repeated values
- Never repeat selectors — add properties to existing ones
- Structure CSS — Reset → Variables → Layout → Components → Utilities
